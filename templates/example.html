How it works

  1. 1

    Install the dependency

    pip install requests
  2. 2

    Point at the Flask server

    By default the server runs on http://localhost:8000. Change the host to match your deployment.

  3. 3

    Run the script

    You'll be prompted for a word. The definition is printed from the JSON response field data['def'].

data_object Expected JSON Response 
{
  "word": "happy",
  "def":  "Feeling or showing pleasure or contentment."
}
python_ex.py 
import requests

user_input = input("Enter a word to look up: ")

api_url = f"http://localhost:8000/api/{user_input}"
response = requests.get(api_url)
data = response.json()

if response.status_code == 200:
    data = response.json()
    print(
        f"The definition of '{data['word']}' is: {data['def']}"
    )

Java Example

The Java integration guide using HttpClient + Gson is in progress.

JavaEx.java 
import java.net.http.*;
import java.net.URI;

public class JavaEx {
    public static void main(String[] args)
        throws Exception {
        String word = "serendipity";
        var client = HttpClient.newHttpClient();
        var request = HttpRequest.newBuilder()
            .uri(URI.create(
              "http://localhost:8000/api/" + word))
            .GET().build();
        var resp = client.send(request,
            HttpResponse.BodyHandlers.ofString());
        System.out.println(resp.body());
    }
}

Quick Reference

route

Endpoint

GET /api/{word}

Replace {word} with any English word. URL-encode spaces if needed.

data_object

Response Shape

{
  "word": "string",
  "def":  "string"
}

Both fields are always strings. def may be null if the word isn't found.

http

Status Codes

  • 200 Word found, definition returned
  • 404 Word not in dictionary CSV
  • 500 Internal server error
